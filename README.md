# Fake Excuse Generator

A Machine Learning based project that generates realistic and creative excuses
using text generation concepts and AI-assisted development.

---

## What it does

- Takes a situation as input (e.g. "late to class", "missed assignment")
- Generates a realistic, context-aware excuse using ML logic
- Demonstrates practical application of basic NLP and text generation concepts

---

## Tech stack

| Area | Technology |
|---|---|
| Language | Python |
| Concepts | Machine Learning, NLP basics, Text Generation |
| Tools | VS Code, Git, AI-assisted development |

---

## How to run it

1. Clone the repository
```bash
git clone https://github.com/chandinipothula99-alt/Fake-Excuse-Generator.git
cd Fake-Excuse-Generator
```

2. Install dependencies
```bash
pip install -r requirements.txt
```

3. Run the project
```bash
python main.py
```
